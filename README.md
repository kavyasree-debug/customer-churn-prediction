# Customer Churn Prediction

## Overview
This project predicts whether a customer will churn using machine learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Models
- Logistic Regression
- Random Forest

## Results
- Accuracy: ~78%
- Key Insight: TotalCharges is a major factor influencing churn

## Workflow
1. Data Cleaning
2. Feature Engineering
3. Model Training
4. Evaluation
5. Prediction System
