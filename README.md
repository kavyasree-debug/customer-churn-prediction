# Customer Churn Prediction

This project provides a machine learning solution to predict customer churn.

## Overview

The goal is to build a predictive model that identifies customers likely to churn, enabling proactive retention strategies.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Models
- Logistic Regression
-Random Forest

## Features

- Data preprocessing and exploration
- Feature engineering
- Model training and evaluation
- Performance metrics and visualization

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python train.py
python predict.py
```

## Results
- Accuracy: ~78%
- Key Insight: TotalCharges is a major factor influencing churn

## Workflow
1. Data Cleaning
2. Feature Engineering
3. Model Training
4. Evaluation
5. Prediction System
